## Description

Populate the Services page with actual content for each service offering. This includes writing the copy, defining each service card's content, and filling in the process/workflow section.

### Service Offerings (5 cards)

1. **Custom Dashboarding**
   - Target: Small and medium-sized businesses
   - Value prop: Transform raw data into visual intelligence your team actually uses
   - Scope: Dashboard design, data pipeline setup, tool selection (Metabase, Grafana, Power BI, custom)

2. **Contract Data Analytics**
   - Target: Teams that need analytical horsepower without a full-time hire
   - Value prop: Deep-dive analysis on your timeline, not your headcount
   - Scope: Exploratory analysis, statistical modeling, reporting, insight synthesis

3. **Data Science**
   - Target: Organizations with data assets they are not leveraging
   - Value prop: ML models and predictive systems that drive decisions
   - Scope: Feature engineering, model development, validation, deployment strategy

4. **AI Enablement**
   - Target: Teams looking to integrate AI into existing workflows
   - Value prop: Practical AI adoption — not hype, not vaporware
   - Scope: RAG systems, agentic architectures, LLM integration, knowledge graphs, MCP

5. **Technology Consulting**
   - Target: Leaders making architectural or strategic technology decisions
   - Value prop: Independent advice from someone who builds, not just advises
   - Scope: Architecture review, tool selection, team structure, AI strategy

### Process Section

Brief 3-step process: Discovery (understand the problem) -> Execution (build the solution) -> Delivery (hand off with documentation and training)
